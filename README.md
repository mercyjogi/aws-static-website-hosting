# AWS Static Website Hosting

## Project Description

This project demonstrates how a static website can be hosted using AWS S3.

The website consists of HTML, CSS, and JavaScript files and is designed as a cloud computing internship project.

---

## Features

- Responsive design
- Interactive button
- Modern landing page
- AWS S3 deployment ready

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- AWS S3 (Deployment Concept)
